\doxysubsection{Detailed Description}
Represents a quadratic equation of the form A \texorpdfstring{$\ast$}{*} X\texorpdfstring{$^\wedge$}{\string^}2 + B \texorpdfstring{$\ast$}{*} X + C = 0. 

This class provides methods to manage quadratic equations, including solving for real roots, operator overloading for equation addition, and safe memory management using unique pointers. It follows the Rule of Three and Rule of Five to handle deep copying and resource ownership transfer efficiently.

Key constraints\+:
\begin{DoxyItemize}
\item {\ttfamily a} (coefficient of X\texorpdfstring{$^\wedge$}{\string^}2) must be non-\/zero; attempts to set {\ttfamily a} to zero will throw an exception.
\item Solutions are calculated dynamically when coefficients are modified. 
\end{DoxyItemize}
