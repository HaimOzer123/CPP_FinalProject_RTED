\doxysubsection{Detailed Description}
Base class representing a general animal. 

The constructor takes {\ttfamily const std\+::string\& name} instead of {\ttfamily std\+::string name} to avoid unnecessary copying. Passing by {\ttfamily const} reference allows efficient access to the original string without creating a duplicate, which is faster and saves memory.
